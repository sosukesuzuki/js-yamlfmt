# js-yamlfmt

JS tools for [yamlfmt](https://github.com/google/yamlfmt).

## packages

| package            | description                                          |
| ------------------ | ---------------------------------------------------- |
| `@js-yamlfmt/node` | CLI tool to format YAML files by yamlfmt via Node.js |
| `@js-yamlfmt/wasm` | yamlfmt JavaScript library                           |
